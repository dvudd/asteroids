# Asteroids
[Asteroids](https://en.wikipedia.org/wiki/Asteroids_(video_game)) clone written in C++ using [SFML](https://www.sfml-dev.org/)

## Build
   ```
   cmake -B build
   cmake --build build
   ```
Note: If you're using NixOS, use the provided flake `nix develop` before building.

The executable is located in build/bin
